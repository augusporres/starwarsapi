## Description

StarWars api project

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

SE PUEDE ACCEDER A LA DOCUMENTACION DE SWAGGER EN LA URL: http://localhost:3000/api/v1/swagger

cualquier usuario registrado se guarda con el rol de 'user'.

# production mode
$ npm run build
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
